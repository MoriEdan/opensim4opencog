COMPILING

1) Compile the FrameNet project as described in README.txt in the FrameNet
   project.

2) Right-click on the TestApplication project, select "Properties", and go to
   "Reference Paths". Add the Public folder that was checked out when building
   the FrameNet project.

3) Right-click on the TestApplication project and select "Build".

Note:  you'll need to adjust the FrameNet file paths in the TestForm
constructor to match your system.


USE

This test application demonstrates some of the basic functionality provided by
the API.
